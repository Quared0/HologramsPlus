# HologramsPlus
Simple and feature-packed Holograms plugin with an advanced API.

# How to access API
Maven:
```
<repositories>
    <repository>
        <id>jitpack.io</id>
        <url>https://jitpack.io</url>
    </repository>
</repositories>

<dependencies>
	<dependency>
		<groupId>com.github.quared0</groupId>
		<artifactId>HologramsPlus</artifactId>
		<version>VERSION</version>
		<scope>provided</scope>
	</dependency>
</dependencies>
```
